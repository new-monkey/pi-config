# pi-config — My Pi Coding Agent Setup

Role manager + self-evolution extension for Pi.

## Install

```bash
pi install git:github.com/ou/pi-config
```

## Roles

| Role | Description |
|------|-------------|
| `dev` | Full development |
| `review` | Read-only code review |
| `debug` | Investigative debugging |
| `arch` | Architecture & design |
| `plan` | Plan before doing |
| `spec` | Spec-driven development |
| `tdd` | Test-driven development |
| `ops` | DevOps / deploy |
| `doc` | Documentation |
| `refactor` | Safe refactoring |
| `explore` | Learning new codebase |
| `novice` | Step-by-step guidance |

## Commands

```
/role          Select a role
/role dev      Switch directly
Ctrl+Shift+R   Cycle roles
/teach         Teach Pi a lesson
/evolve stats  Show evolution stats
```
